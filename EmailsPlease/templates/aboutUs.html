About This Project

This project was created as part of the Comp Sci 372 Capstone at the University of Wisconsin–Green Bay. The goal is to design and develop a meaningful software application that demonstrates both technical skill and real-world value.

We focused on Cybersecurity Awareness—helping users identify phishing attempts, spam, and legitimate emails. This game provides a safe, interactive environment where players practice spotting suspicious messages and build confidence in navigating digital communication.
