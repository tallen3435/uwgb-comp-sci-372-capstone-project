About This Project

This project was created as part of the Comp Sci 372 Capstone at the University of Wisconsin–Green Bay. The goal of the capstone is to design and develop a meaningful software application that demonstrates both technical skill and real-world value.

For this project, we chose to focus on Cybersecurity Awareness—specifically, helping users learn how to identify phishing attempts, spam messages, and legitimate emails. Cyber threats continue to grow more sophisticated, and many successful attacks rely on tricking everyday users rather than exploiting technical vulnerabilities.

This game provides a safe, interactive environment where players can practice spotting suspicious messages, learn common red flags, and build confidence in navigating digital communication. Instead of passively reading about cybersecurity, users learn by doing—making decisions, reviewing simulated emails, and experiencing the consequences in a controlled setting.

The project reflects the culmination of coursework in software engineering, web development, and cybersecurity fundamentals. It showcases the importance of creating tools that are not only technically sound, but also accessible, educational, and genuinely useful for everyday users.
